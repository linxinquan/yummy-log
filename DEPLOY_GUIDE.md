# 觅食图小程序 - 上线配置指南

> 本文档说明小程序上线微信平台前需要在微信公众平台配置的内容

## 1. 域名白名单配置

在微信公众平台 → 开发管理 → 开发设置 → 服务器域名中配置以下域名：

### request合法域名（必须）
```
https://apis.map.qq.com
```

> **注意**：地图Key使用的是腾讯位置服务，调用逆地址解析API需要在request域名中添加 `apis.map.qq.com`

### downloadFile合法域名（如使用）
如果小程序需要下载图片资源，需要配置图片域名：
```
https://img.btpig.cn  (示例，需根据实际图片存储位置配置)
```

## 3. 隐私权限说明

当前小程序已正确配置以下隐私权限声明：

| 权限 | 声明内容 | 状态 |
|------|----------|------|
| getLocation | 你的位置信息将用于展示附近美食 | ✅ |
| chooseLocation | 选择位置用于店铺定位 | ✅ |

这些权限在小程序后台已配置，如有问题可在审核备注中说明使用场景。

## 5. 上传步骤

1. 打开微信开发者工具
2. 点击右上角"上传"按钮
3. 填写版本号：`1.0.0`
4. 填写版本说明：`深圳蛇口美食探店地图小程序`
5. 提交审核

---

## 6. 审核常见问题

### 6.1 位置权限
小程序使用位置权限已正确声明，如被拒绝可申诉说明：用于展示附近美食店铺位置

### 6.2 地图展示
使用微信原生Map组件展示地图，无需额外资质

### 6.3 用户头像昵称
"我的"页面使用微信开放能力获取头像昵称，已正确配置

---

## 7. 常见错误排查

| 错误 | 原因 | 解决方案 |
|------|------|----------|
| 定位失败 | 未授权位置权限 | 在手机设置中开启位置权限 |
| 地图加载失败 | Key配置错误或域名未添加 | 检查Key和域名配置 |
| request fail | 域名未在白名单 | 在公众平台添加域名 |
