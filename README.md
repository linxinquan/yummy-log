# 觅食图 / Me Tour - 微信小程序版

> 深圳蛇口美食探店地图 · 微信小程序

## 功能特性

- 🗺️ **地图浏览** - 微信原生地图，展示蛇口地区美食店铺
- 🔍 **店铺搜索** - 按分类、评分、价格筛选
- ❤️ **想去功能** - 收藏喜欢的店铺
- 📝 **个人记录** - 记录专属美食回忆
- 🗺️ **路线规划** - 贪心最近邻算法优化游览路线
- 📍 **多地图导航** - 支持高德、百度、腾讯、苹果地图
- 📝 **攻略导入** - 粘贴小红书/大众点评攻略文本，智能识别店铺
- ➕ **添加店铺** - 发现未收录的新店可分享到地图

## 快速开始

### 1. 获取 AppID

1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 注册小程序账号
3. 获取 AppID

### 2. 导入项目

1. 下载微信开发者工具
2. 新建项目，选择「导入项目」
3. 选择本项目目录，填入 AppID

### 3. 配置云开发（可选）

1. 在微信开发者工具中开通云开发
2. 创建云环境
3. 在 `app.js` 中填入云环境 ID

```javascript
// app.js
cloudEnv: 'your-cloud-env-id'
```

### 4. 运行调试

- 点击「编译」运行小程序
- 使用「真机调试」测试完整功能

## 数据说明

- **初始数据**: 16 家蛇口美食店铺（硬编码在 `utils/shopData.js`）
- **用户添加**: 存储在 localStorage，key = `userAddedShops`
- **用户数据**: 想去/觅食记录存储在 localStorage，key = `userShops`
- **攻略数据**: 存储在 localStorage，key = `myGuides`

## 地图 Key

需要在 `app.js` 中配置腾讯地图 Key：

```javascript
qqMapKey: 'YOUR_QQMAP_KEY'
```

申请地址：[腾讯位置服务](https://lbs.qq.com/)
